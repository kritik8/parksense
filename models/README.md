# models/

This directory stores trained ML model artifacts (gitignored).

| File | Description |
|------|-------------|
| `hotspot_lgbm.pkl` | LightGBM hotspot forecasting model |
| `cis_normalization.pkl` | CIS normalization parameters |
| `blr_graph_centrality.pkl` | Pre-computed edge betweenness centrality |

Train the models by running the notebooks in `notebooks/`.
